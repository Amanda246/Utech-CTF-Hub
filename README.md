# UTech CTF Hub
**CNS3005 Ethical Hacking — Assignment 2**

A Capture The Flag learning platform for cybersecurity education.

## Features
- 18 CTF Challenges (Beginner / Intermediate / Advanced)
- 6 Categories: Web Hacking, Cryptography, Network Security, Reverse Engineering, Social Engineering, Data Security
- 3 Knowledge Quizzes (30 questions total)
- Jeopardy Board (25 questions, 5 categories)
- 8 Learn Topics with full content
- Scoreboard
- Interactive SQL Injection & XSS Training Pages
- Admin Dashboard

## Deployment
This site is deployed via GitHub Pages. All data is stored in browser localStorage — no backend required.

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Bootstrap 5.3
- Bootstrap Icons
- Web Crypto API (SHA-256 for password hashing)
